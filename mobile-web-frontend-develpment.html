MADE US THEIR PARTNERS

Successful client cases that outsourced with us

Flipp

Flipp is a digital marketplace used by millions that reinvented how people plan their weekly shopping trips, allowing customers to decide what to buy based on proximity, best cost, and other filters across North America.

Since Flipp acquired Reebee in 2022, the Channel Experience team has started working on merging similar functionalities in both apps under Flipp’s design and improving the features. Using Java, Kotlin, MVVM, MVP, and Gradle, the team is creating Flipp's clone app, reusing and adapting the current code, and incorporating Reebee's design into it.


Feeld

Feeld is an inclusive dating app and global community for both singles and couples to explore their desires and find meaningful connections.

The frontend team started working on the chat refactor for the Feeld mobile app. That chat was implemented based on the Stream Chat API and infrastructure and by managing and filtering the Chat API payload with our logic and presenting it in the UI with our React Native components.

However, after speaking with the Stream group and discussing the pros and cons, we realized that we could improve the app performance and save time by implementing several functionalities using their Stream Chat React Native SDK which contains fully customizable components. We already refactored the Chat and implemented that SDK and that improvement is available to users in the store.


Confidential

This emerging and leading mental health tech company offers a mobile application that pairs pediatric sleep experts with AI to create a custom sleep plan for infants and toddlers.

Our team is now working on a new app that works as a timer to address specific scenarios such as ways to properly prepare for bedtime, and also by providing parents with the motivation and confidence to get started, with easy-to-follow instructions and personalized plans based on each specific scenario. The app is being developed entirely in React Native and Firebase.

As for their main sleep tracking app, we’re helping with migrating the server to a new architecture organized on modules for the different cloud functions on Firebase and adding more test coverage to the code.


MyFieldAudits

MyFieldAudits is a digital on-site inspection software used in care facilities that helps users complete field audits using an app developed for mobile devices.

Our unique challenge was to completely replace their legacy system with the development from scratch of a new optimized app that included a backend, a web frontend, a back office, and two mobile applications, performing a corresponding data ETL analysis and executing the migration of user information and statistical data to the new system without interrupting the service for the end user. This was achieved with a team lead on our end, managing a team of four developers.
